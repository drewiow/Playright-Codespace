# 🌍 Find Sites On All Regions

This automation checks whether a provider exists across all configured system regions.

---

## 🔍 What this script does

- Logs into each configured region in turn  
- Searches for a provider using the supplied **ODS Code**  
- Determines whether the provider exists in each region  
- Logs the outcome for each region  

## ⚙️ Requirements

Before running this script, you must provide:

- ✅ Your encrypted environment file (`.enc`)
- ✅ A valid **ODS Code**

---

## 🚫 What this script does NOT do

- It does **not modify any data**
